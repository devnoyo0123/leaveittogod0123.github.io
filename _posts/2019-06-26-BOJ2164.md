# 목표: 백준 2164 풀기
### 환경: mac OS Mojave 10.14, CLion

### 로직

#### 문제이해

설명해준대로 구현하라 -> 시뮬레이션

#### 문제접근

N제한이 50만임. -> ( N^2 >> 1억 )  -> 시간초과


~~~c++
#include <iostream>
#include <deque>

using namespace std;

int main() {

    int n;
    cin >> n;
    deque<int> dq;
    for(int i=1; i<=n; i++){
        dq.push_back(i);
    }

    while(dq.size()>1){
        dq.pop_front();
        if( dq.size() == 1){
            break;
        }
        int front = dq.front(); dq.pop_front();
        dq.push_back(front);
    }

    cout<<dq.front();

    return 0;
}
~~~
